Política de Privacidad - MaikPet

Última actualización: 16 de Abril de 2026

1. Datos Recolectados

MaikPet recolecta los siguientes datos del usuario:

  • Información de registro: nombre completo, dirección, teléfono, email y edad
  • Edad: verificada para cumplir con el requisito de edad mínima de 18 años
  • Usage data: interacciones dentro de la aplicación para adopción de animales

2. Finalidad del Tratamiento de Datos

Los datos recolectados tienen como única finalidad:

  • Establecer conexión entre usuarios para la adopción responsable de animales
  • Facilitar el proceso de adopción entre particulares
  • Gestionar el registro y autenticación de usuarios
  • Mantener comunicación relacionada con el servicio de adopción

3. Retención de Datos

Los datos del usuario serán conservados por un período de 90 días desde la fecha de registro.

Después de este período, los datos serán eliminados automáticamente del sistema.

4. Eliminación de Datos

El usuario puede solicitar la eliminación de sus datos en cualquier momento mediante:

  • Uso de la opción "Cerrar Sesión" dentro de la aplicación
  • Contacto directo con soporte

La eliminación incluye: perfil de usuario, mascotas publicadas y cualquier dato asociado.

5. Limitación de Responsabilidad

MaikPet actúa únicamente como intermediaria entre usuarios para la adopción de animales.

La plataforma no garantiza:

  • El comportamiento de terceros usuarios
  • La disponibilidad de animales publicados
  • La veracidad de la información proporcionada por usuarios
  • El resultado de las transacciones de adopción

Los usuarios son responsables de verificar la información y tomar sus propias decisiones.
